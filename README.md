## HTML tags

The css file is given to you, you should use the following HTML tags:

### Block level tags
* `<header>`
* `<nav>`
* `<ul>`
* `<li>`
* `<main>`
* `<section>`
* `<h2>`
* `<p>`
* `<footer>`
* `<address>`
* `<div>`

### Inline level tags
* `<a>`
* `<small>`
* `<em>`
